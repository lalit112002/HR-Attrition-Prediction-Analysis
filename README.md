# HR-Attrition-Prediction-Analysis
Project Overview 
Developed an Data Science project to analysis or predicting employee attrition rate
• It offers dynamic data from IBM to predict that which Employee is leaving the company and from which department in
future for several reasons like Environmental satisfaction, Salary reasons, Distance from company, educational field
• Applied Data Mining and statistics, Data visualization and various algorithm techniques to develop the project
Goal-
The goal of this project is to analyze employee data and build a predictive model that identifies the factors influencing employee attrition (resignation/turnover) and predicts which employees are at risk of leaving the organization.

Problem Statement:
Employee attrition can significantly affect an organization’s performance, increasing recruitment and training costs. This project aims to help the HR department proactively retain employees by using data-driven insights to understand key attrition factors.

Tools & Technologies Used:
Programming Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
Machine Learning Models: Logistic Regression, Decision Tree, Random Forest, XGBoost
Environment: Jupyter Notebook / Google Colab
